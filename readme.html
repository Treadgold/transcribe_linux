Real-Time Whisper Transcription

Optimized for WSL2 & Native Linux

A high-performance Python tool providing a live-updating terminal dashboard for transcribing microphone audio in real-time using OpenAI's Whisper.

‚ú® Features

  • Rich Terminal UI: Live VU meter, status updates, and word-wrapped transcription area.
  • Smart Backend Switching: Automatically detects WSL2 (PulseAudio) vs Native Linux (SoundDevice).
  • Streaming Transcription: Displays text immediately as you speak.
  • Smart File Management: Auto-saves to the transcripts/ folder with unique, non-overwriting filenames.
  • Clean Exit: Press ENTER to stop gracefully.

üí° WSL2 Audio Tips

This script is optimized for WSLg (Windows 11). If your Windows mic is set as the "Default Communication Device," WSL2 sees it automatically via PulseAudio.

  • If the VU meter isn't moving, check that your mic isn't muted in Windows.
  • No physical USB passthrough (usbipd) is required for most setups.

‚ö†Ô∏è Troubleshooting

  • No text appearing? Check the "Mic Volume" bar. If it isn't moving, the system isn't receiving audio.
  • Slow processing? Large models require a GPU or powerful CPU. If the status hangs on "Transcribing", use -m tiny.
  • Permissions? Ensure your user is in the audio group: sudo usermod -aG audio $USER (then log out and back in).
